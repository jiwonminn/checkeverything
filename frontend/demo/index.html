CheckEverything Screenshot Demo

Local testing pages for capturing README demo assets. Uses DEMO_MODE — no live API calls required.

Before you start

  1. Run ./scripts/demo-screenshots.sh
  2. Reload the extension in chrome://extensions
  3. Set extension API URL to http://localhost:8080
