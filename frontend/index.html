5-agent code review

Paste code or a PR diff on the left to get started.

Five specialist agents review your submission in parallel. A coordinator synthesizes scores, findings, and prioritized action items.

  • Security — secrets, injection, unsafe patterns
  • Correctness — logic bugs and edge cases
  • Readability — naming, structure, clarity
  • Performance — inefficiencies and bottlenecks
  • Tests — coverage gaps and testability
  1. Click Load sample
  2. Press Run 5-Agent Review
  3. Review scores and findings here
  4. Adjust Agent score weights if you want (Security defaults to 20%)

AI answer trust check

Paste an AI answer on the left to check its claims.

Checks whether claims are supported by cited sources, evaluates source quality, and flags weak or unsupported statements — the same analysis as the Chrome extension.

  • Claim support — are statements backed by evidence?
  • Source quality — are citations from reliable domains?
  • Citation accuracy — do sources actually say what the AI claims?
  • Freshness — could the information be outdated?
  • Bias / context — is anything one-sided or missing?
  1. Click Load sample
  2. Press Check Trust Score
  3. See score, claims, and source checks here
