# Demo Assets

These files power the README **Demo** section. Capture them locally — they are not generated by CI.

## Required screenshots

| File | What to capture |
| --- | --- |
| `chatgpt-badge.png` | ChatGPT answer with **Trust Score** badge visible |
| `google-ai-overview-badge.png` | Google Search **AI Overview** with badge |
| `claim-panel-statuses.png` | Detail panel open — overall score, sources, colored claims |
| `options-weights.png` | Extension options page with trust score weights |

## How to capture

**Fastest path — local screenshot demo (no ChatGPT/Google login required):**

```bash
./scripts/demo-screenshots.sh
```

Then reload the extension and open http://localhost:8080/demo

**Or use live sites:**
2. Reload extension in `chrome://extensions`
3. Use ChatGPT answer **with citations** for best claim matching
4. Search Google for a query that triggers **AI Overview**
5. Click **Trust Score** before screenshotting the panel

Full video flow: [`DEMO_SCRIPT.md`](DEMO_SCRIPT.md)
